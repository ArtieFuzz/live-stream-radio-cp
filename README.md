# live-stream-radio-cp
Simple web control panel for live-stream-radio.
Any PR is welcome.

# Installation
- Clone this repo.
- Replace `http://localhost:8000` and `testkey` in the `config.json` with your url and api key.
```
npm install
npm start
```
# Usage
Just open `http://localhost:3000`
